# examen_frontend-develop

## Project setup
```
npm install
```

## install vue
```
npm install -g @vue/cli
```

## install vue-router
```
npm install vue-router
```
## install axios
```
npm install axios
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
